# Chetan's Dokploy Templates

Personal Dokploy template registry for the Zimablade homelab.

## Templates

- **supabase-db** — the official Dokploy Supabase template plus:
  - `kong` joins the external `dokploy-network`, so Traefik can always reach it
  - `traefik.docker.network=dokploy-network` label, so Traefik picks the right network IP

## Usage

1. Push this repo to GitHub (public), e.g. `chetangk/dokploy-templates`.
2. In Dokploy: **Create Service → Template**, open the template-source/base-URL setting and set it to:

```
https://raw.githubusercontent.com/chetangk123/dokploy-templates/main
```

3. Pick **Supabase (Homelab)** and deploy. Domains, JWT secret, anon/service keys, and passwords are auto-generated by `template.toml`.

## Adding more templates

Copy a folder from the official repo (https://github.com/Dokploy/templates) into `blueprints/<id>/`,
adjust `meta.json` (unique `id`), apply your changes, and append the same meta object to the root `meta.json`.
